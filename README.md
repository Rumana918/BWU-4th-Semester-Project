# LearnSphere

##  About the Project

LearnSphere is a modern educational web portal designed to help students explore colleges, courses, entrance examinations, and academic resources from one platform.

The project provides a clean and responsive user interface where students can browse educational categories, search for institutions, and access useful academic information.

This project was developed using HTML, CSS, JavaScript, Bootstrap, and is intended to be upgraded into a full-stack PHP and MySQL application.

---

## ✨ Features

* Responsive Homepage
* Modern Navigation Bar
* Hero Image Carousel
* College Search Section
* College Recommendation Cards
* Multiple Academic Categories
* Login Page
* Student Registration Page
* About Us Page
* Contact Us Page
* Mobile Responsive Design
* Local Storage based Demo Authentication

## ⚙ Installation

1. Download or Clone the repository.

```
git clone https://github.com/Rumana918/BWU-4th-Semester-Project/
```

2. Move the project into the XAMPP `htdocs` directory.

3. Start Apache and MySQL.

4. Import the database.

5. Open the browser.

```
http://localhost/LearnSphere
```
